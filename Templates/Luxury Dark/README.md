# Luxury Dark Photography Template

## Premium Cinematic Photography Website

A sophisticated, high-end photography template with a dark cinematic aesthetic. Perfect for fine art photographers, luxury brands, and premium creative studios.

### 🎨 Design Features
- **Black & Gold Styling**: Elegant color palette
- **Serif Typography**: Premium, sophisticated fonts
- **Cinematic Hero**: Large dramatic images
- **Premium Branding**: Photographer name & logo support
- **Smooth Animations**: Elegant transitions
- **Mobile Optimized**: Fully responsive design

### 📋 Pages Included
- `index.html` - Hero landing page with gallery preview
- `gallery.html` - Full portfolio gallery with lightbox
- `contact.html` - Contact form & inquiries
- `assets/` - CSS, JavaScript, image placeholders

### 🎯 Customization Guide

#### Change Photographer Name
In `index.html`, find:
```html
<!-- CUSTOMIZE: Change photographer name here -->
<h1 class="photographer-name">Your Name Here</h1>
```

#### Add Your Logo
In `index.html`, find:
```html
<!-- CUSTOMIZE: Replace with your logo image -->
<img src="assets/images/logo.png" alt="Logo" class="logo">
```

#### Replace Gallery Images
In `gallery.html`, replace image paths:
```html
<!-- CUSTOMIZE: Add your images here -->
<img src="assets/images/photo-1.jpg" alt="Portfolio">
```

#### Edit Colors
In `assets/css/style.css`, modify the root variables:
```css
:root {
  --gold: #D4AF37;
  --dark: #1a1a1a;
  --accent: #f0f0f0;
}
```

#### Update Contact Email
In `contact.html`, find:
```html
<!-- CUSTOMIZE: Replace with your email -->
<form action="mailto:your-email@example.com">
```

#### Add Social Links
In footer, update:
```html
<!-- CUSTOMIZE: Add your social media links -->
<a href="https://instagram.com/yourprofile">Instagram</a>
```

### 🚀 Deployment

1. **Replace all placeholder images** with your photography
2. **Update photographer name and bio** in all pages
3. **Add your contact email** in contact form
4. **Update social media links** in footer
5. **Deploy to hosting** (Netlify, Vercel, GitHub Pages, etc.)
