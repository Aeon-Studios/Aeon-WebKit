# Aeon WebTemplates 📸

A premium collection of professional photography website templates for photographers, studios, and creative businesses.

## 🎨 Template Collection

### 1. **Luxury Dark** - Cinematic & Premium
Black and gold aesthetic for high-end photography. Perfect for fine art photographers and luxury brands.
- **Location:** `Templates/Luxury Dark/`
- **Features:** Elegant serif typography, smooth animations, premium branding
- **Interactive Edition:** Full backend with user accounts, favorites, admin dashboard

### 2. **Minimal Light** - Clean & Editorial
Bright, minimalist design focusing on photography. Ideal for editorial and lifestyle work.
- **Location:** `Templates/Minimal Light/`
- **Features:** Lots of whitespace, thin fonts, high contrast
- **Interactive Edition:** Available with enhanced features

### 3. **Agency Bold** - Creative Studio
Large bold typography with animations. Perfect for creative agencies and studio collectives.
- **Location:** `Templates/Agency Bold/`
- **Features:** Strong CTAs, interactive elements, studio vibe
- **Interactive Edition:** Client portals, team profiles, project galleries

### 4. **Nature** - Earthy & Organic
Landscape-focused design with organic elements. For nature and outdoor photographers.
- **Location:** `Templates/Nature/`
- **Features:** Earthy colors, parallax effects, landscape showcase
- **Interactive Edition:** Booking calendar, print ordering, location galleries

### 5. **Wedding** - Romantic & Elegant
Soft, romantic aesthetic for wedding photographers.
- **Location:** `Templates/Wedding/`
- **Features:** Elegant serif fonts, soft pastels, package showcase
- **Interactive Edition:** Booking system, package selection, client galleries

### 6. **Urban** - Edgy & Bold
Dark, contemporary design for street and urban photographers.
- **Location:** `Templates/Urban/`
- **Features:** High contrast, bold grids, gritty aesthetic
- **Interactive Edition:** Collections, like/star system, series organization

### 7. **Portfolio** - Simple & Personal
Clean, focused design for solo creators.
- **Location:** `Templates/Portfolio/`
- **Features:** Bio-focused, skill showcase, personal branding
- **Interactive Edition:** User accounts, collections, download options

### 8. **Store** - eCommerce
Complete print and product store template.
- **Location:** `Templates/Store/`
- **Features:** Product cards, pricing display, cart-ready
- **Interactive Edition:** Full shopping cart, payment integration, inventory management

## 🚀 Quick Start

### Static Templates
Each template is ready to deploy as-is:

1. Choose your template
2. Replace placeholder images with your photography
3. Edit photographer name and contact info
4. Update social media links
5. Deploy to Netlify, Vercel, or GitHub Pages

### Interactive Editions with Backend
For advanced features, use the interactive editions:

1. Navigate to template's `Interactive Design/` folder
2. Deploy frontend to hosting (Netlify, Vercel, etc.)
3. Set up backend (Java, PHP, Node.js)
4. Connect frontend to backend API
5. Configure database and authentication

## 🛠️ Deployment

### Static Templates
**Easiest option - No backend required**

Deploy to:
- Netlify (recommended)
- Vercel
- GitHub Pages
- Traditional web hosting
- Any static hosting service

Steps:
1. Upload files to hosting
2. Configure domain
3. Done!

### Interactive Editions
**Full-featured with backend**

Frontend:
1. Deploy to Netlify/Vercel/GitHub Pages

Backend:
1. Choose your stack (Java/PHP/Node.js/Python)
2. Set up database (SQLite/MySQL)
3. Deploy to:
   - Heroku (simple)
   - AWS (scalable)
   - DigitalOcean (affordable)
   - Railway (modern)
   - Your own server

## 🔧 Technical Details

### Frontend Stack
- **HTML5** - Semantic markup
- **CSS3** - Modern responsive styling
- **Vanilla JavaScript** - No dependencies
- **Responsive Design** - Mobile-first approach

### Backend Stack (Interactive)
- **Java:** Spark Java, SQLite, Gson
- **PHP:** PHP 7+, SQLite, PDO
- **Node.js:** Express, SQLite3, JWT
- **Python:** Flask, SQLite, Bcrypt

### Database
- **SQLite** - Recommended (zero setup)
- **H2** - Java in-memory option
- **MySQL** - For larger deployments

### Security
- Password hashing (bcrypt)
- Input validation
- CORS protection
- Session/JWT authentication
- Admin verification
- Upload restrictions
