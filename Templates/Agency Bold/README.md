# Agency Bold Photography Template

## Creative Studio & Agency Portfolio

Bold, modern template with large typography and interactive animations. Perfect for creative agencies, studio collectives, and bold visual brands.

### 🎨 Design Features
- **Large Bold Text**: Typography as design
- **Smooth Animations**: Interactive elements
- **Strong CTA Sections**: Clear calls-to-action
- **Creative Grid Layouts**: Modern grid design
- **Studio Vibe**: Collaborative energy
- **Dynamic Interactions**: Hover effects & transitions

### 📋 Pages Included
- `index.html` - Bold hero with featured projects
- `gallery.html` - Animated portfolio grid
- `contact.html` - Engagement form
- `assets/` - CSS, JavaScript, image placeholders

### 🎯 Customization Guide

#### Studio/Agency Name
In `index.html`:
```html
<!-- CUSTOMIZE: Your studio name -->
<h1 class="studio-name">Studio Name</h1>
```

#### Add Logo
```html
<!-- CUSTOMIZE: Your logo here -->
<img src="assets/images/logo.png" class="studio-logo">
```

#### Portfolio Projects
In `gallery.html`:
```html
<!-- CUSTOMIZE: Add your projects -->
<div class="project">
  <img src="assets/images/project-1.jpg">
  <h3>Project Name</h3>
</div>
```

#### Modify Colors
In `assets/css/style.css`:
```css
:root {
  --primary: #000;
  --accent: #FF6B35;
  --bg: #f5f5f5;
}
```

#### Update Services
In `index.html`:
```html
<!-- CUSTOMIZE: List your services -->
<div class="service">Photography</div>
<div class="service">Videography</div>
```

#### Contact Email
In `contact.html`:
```html
<!-- CUSTOMIZE: Your email -->
<form action="mailto:hello@studio.com">
```
