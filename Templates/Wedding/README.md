# Wedding Photographer Template

## Elegant, Romantic Photography Portfolio

Sophisticated wedding photography template with soft aesthetics and elegant styling. Perfect for wedding photographers and romantic event specialists.

### 🎨 Design Features
- **Romantic Elegance**: Soft, refined aesthetic
- **Soft Typography**: Elegant serif fonts
- **Pastel Tones**: Soft color palette
- **Luxury Feel**: Premium presentation
- **Smooth Animations**: Gentle transitions
- **Responsive Design**: Mobile optimized

### 📋 Pages Included
- `index.html` - Romantic hero landing
- `gallery.html` - Wedding portfolio gallery
- `contact.html` - Inquiry & booking form
- `assets/` - CSS, JavaScript, image placeholders

### 🎯 Customization Guide

#### Your Name/Studio
In `index.html`:
```html
<!-- CUSTOMIZE: Your name -->
<h1 class="photographer-name">Wedding Photographer</h1>
```

#### Logo/Branding
```html
<!-- CUSTOMIZE: Upload logo -->
<img src="assets/images/logo.png" class="logo">
```

#### Portfolio Images
In `gallery.html`:
```html
<!-- CUSTOMIZE: Wedding photos -->
<img src="assets/images/wedding-1.jpg" class="wedding-photo">
```

#### Color Palette
In `assets/css/style.css`:
```css
:root {
  --primary: #8B4789;
  --secondary: #D4A5A5;
  --accent: #f5e6e6;
  --text: #2c2c2c;
}
```

#### Bio/About
In `index.html`:
```html
<!-- CUSTOMIZE: Your story -->
<p class="bio">I capture love stories...</p>
```

#### Services Offered
In `index.html`:
```html
<!-- CUSTOMIZE: Your packages -->
<div class="service">
  <h3>Premium Package</h3>
  <p>Full day coverage</p>
</div>
```

#### Contact & Inquiries
In `contact.html`:
```html
<!-- CUSTOMIZE: Your email -->
<form action="mailto:weddings@example.com">
```

#### Social Links
```html
<!-- CUSTOMIZE: Your social profiles -->
<a href="https://instagram.com/yourprofile">Instagram</a>
```
