# Minimal Light Photography Template

## Clean Editorial Photography Website

A bright, modern minimalist template with editorial style. Ideal for lifestyle photographers, editorial work, and minimalist creative brands.

### 🎨 Design Features
- **Bright White Space**: Clean modern aesthetic
- **Thin, Elegant Fonts**: Editorial typography
- **Lots of Breathing Room**: Generous spacing
- **High Contrast Images**: Bold photo showcase
- **Minimal UI**: Focus on photography
- **Fully Responsive**: Mobile-first design

### 📋 Pages Included
- `index.html` - Clean landing with featured work
- `gallery.html` - Minimal gallery grid
- `contact.html` - Simple contact form
- `assets/` - CSS, JavaScript, image placeholders

### 🎯 Customization Guide

#### Change Photographer Name
In `index.html`:
```html
<!-- CUSTOMIZE: Replace with your name -->
<h1>Your Name</h1>
```

#### Add Logo
In `index.html`:
```html
<!-- CUSTOMIZE: Upload your logo -->
<img src="assets/images/logo.svg" alt="Logo">
```

#### Replace Photos
In `gallery.html`:
```html
<!-- CUSTOMIZE: Replace with your images -->
<img src="assets/images/photo-1.jpg" class="gallery-image">
```

#### Edit Typography
In `assets/css/style.css`:
```css
body {
  font-family: 'Thin Font Name', sans-serif;
  letter-spacing: 0.1em;
}
```

#### Update Bio
In `index.html`:
```html
<!-- CUSTOMIZE: Write your bio here -->
<p class="bio">Your biography...</p>
```

#### Add Email to Contact Form
In `contact.html`:
```html
<!-- CUSTOMIZE: Your email here -->
<form action="mailto:you@example.com">
```
