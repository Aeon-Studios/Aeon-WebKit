# Photography Template

A modern photography website template designed for photographers, studios, creators, and visual brands who want a clean professional online presence.

## Overview

This template includes a top banner/header section featuring:

- Photographer's name  
- Optional logo  
- Navigation menu  
- Favicon support (use your logo as the browser tab icon)

Perfect for portfolios, client galleries, events, prints, and photography businesses.

## Features

- Clean modern layout  
- Mobile responsive design  
- Portfolio/gallery sections  
- Hero banner with branding  
- Easy customization  
- Fast loading structure  
- Professional presentation

## Branding Options

Customize the template by changing:

- Photographer name  
- Logo image  
- Favicon  
- Colors  
- Fonts  
- Gallery images  
- Contact details

## Versions Included

### 1. Standard Version

A clean static version ideal for:

- Personal portfolios  
- Showcase websites  
- Event photographers  
- Studio landing pages

### 2. Interactive Edition (`/interactive-edition`)

A second version located in a subfolder that includes advanced interactive features such as:

- User accounts  
- Favorites / saved photos  
- Star or like system  
- Purchase photos / digital downloads  
- Client photo selections  
- Stored user data  
- Management tools

Ideal for photographers who want client interaction or online sales.
