# Nature & Outdoor Photography Template

## Earthy, Organic Photography Website

Nature-inspired template with earthy tones and organic design elements. Perfect for landscape photographers, outdoor specialists, and nature-focused brands.

### 🎨 Design Features
- **Earthy Color Palette**: Greens, browns, natural tones
- **Landscape Focus**: Full-width hero images
- **Organic Design**: Natural, flowing layouts
- **Nature Elements**: Subtle background textures
- **Smooth Parallax**: Depth and movement
- **Mobile Ready**: Responsive design

### 📋 Pages Included
- `index.html` - Natural hero with landscape feature
- `gallery.html` - Landscape gallery showcase
- `contact.html` - Inquiry form
- `assets/` - CSS, JavaScript, image placeholders

### 🎯 Customization Guide

#### Photographer Name
In `index.html`:
```html
<!-- CUSTOMIZE: Your name -->
<h1 class="photographer-name">Nature Photographer</h1>
```

#### Logo/Branding
```html
<!-- CUSTOMIZE: Upload your logo -->
<img src="assets/images/logo.png">
```

#### Portfolio Images
In `gallery.html`:
```html
<!-- CUSTOMIZE: Add landscape images -->
<img src="assets/images/landscape-1.jpg" class="landscape">
```

#### Color Scheme
In `assets/css/style.css`:
```css
:root {
  --primary: #2d5016;
  --accent: #8b7355;
  --light: #f4f1de;
}
```

#### Bio Section
In `index.html`:
```html
<!-- CUSTOMIZE: Write about your nature photography -->
<p class="bio">I specialize in...</p>
```

#### Specialties
In `index.html`:
```html
<!-- CUSTOMIZE: Your specialties -->
<div class="specialty">Landscape</div>
<div class="specialty">Wildlife</div>
<div class="specialty">Outdoor</div>
```

#### Contact Form
In `contact.html`:
```html
<!-- CUSTOMIZE: Your email -->
<form action="mailto:photos@example.com">
```
